# CodeMentor AI - Landing Page

## Overview

CodeMentor AI is a modern and responsive landing page designed for students and developers who want to improve their coding skills with the help of Artificial Intelligence.

The website showcases the platform's key features, pricing plans, contact information, and user-focused design. It is built using HTML, CSS, and JavaScript.

---

## Features

* Responsive Design
* Modern UI with Gradient Background
* Interactive Navigation Bar
* Hero Section with Statistics
* Feature Cards with Hover Effects
* Pricing Section
* Contact Section
* Smooth Scrolling
* Mobile-Friendly Layout
